# Teknav Frontend (Next.js)

این برنامه همان رابط کاربری Next.js است که در ریشهٔ مخزن قرار دارد. برای سهولت تنظیمات محیطی، فایل `.env.local.example` در این پوشه قرار گرفته است.

## متغیرهای محیطی
- `NEXT_PUBLIC_BACKEND_URL` آدرس سرویس NestJS
- `NEXT_PUBLIC_SITE_URL` دامنه سمت کاربر

## اجرای محلی
```bash
npm install
npm run dev
```
(از ریشهٔ مخزن اجرا کنید تا بسته‌های موجود استفاده شوند.)

## ارتباط با بک‌اند
تمام درخواست‌های داشبوردها و فرم‌ها باید از این متغیر برای فراخوانی API استفاده کنند.
